# solitari
Joc del solitari fet amb C++

Cada carta es representa per dos caràcters, que són ** si està
amagada, i pel seu valor i el coll quan es visible. Els valors de les cartes són els que tenen marcats,
excepte en el cas del 10 que se substitueix per ‘D’. Els colls s’identifiquen per lletres, P: piques; 
c: cors, d: diamants, T: trèvols. Per ajudar a identificar millor les cartes, les lletres són majúscules en
els colls negres i minúscules en els colls vermells.

El tauler de joc disposa d’una capçalera a on es posen
la numeració de les diferents columnes, c1..c7, i al costat de cada columna es posa la numeració de
les files, f1..f7.
